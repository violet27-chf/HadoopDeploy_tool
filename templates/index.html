🌐 ChmlFrp端口映射教程 （公网部署必看）

  1. 注册账号：访问 ChmlFrp官网
  2. 下载客户端：选择对应系统版本
  3. 创建隧道：配置参数：
    • 本地IP：虚拟机内网IP
    • 本地端口：22
    • 隧道类型：TCP
    • 远程端口：自动分配或自定义
    • 中转节点：选择延迟低的节点
  4. 下载配置：获取 frpc.ini
  5. 启动客户端：
    frpc.exe -c frpc.ini
  6. 公网连接：
    ssh 用户名@公网IP -p 端口
注意事项：
  • 检查防火墙和端口占用
  • 选择延迟低的中转节点
  • 更多帮助：官方文档

🌐 公网部署

  • 🔧 需要配置端口映射
  • 🔧 网络延迟相对较高
  • 🔧 功能可能受限

  • 🔧 依赖第三方服务
  • 🔧 可能存在安全风险
适用场景：远程部署、演示展示、临时使用
💡 推荐建议
优先选择本地部署体验完整功能，公网部署适合远程场景。本地部署提供更稳定、更快速、更安全的部署体验。

常见问题

如何激活虚拟环境？

Windows: venv\Scripts\activate
Linux: source venv/bin/activate

依赖安装失败怎么办？

升级pip，或使用国内镜像源：
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/

端口被占用如何处理？

使用 netstat -ano | findstr :5000 查找占用进程，或修改app.py端口。

如何在frp中配置端口映射服务？

推荐使用 ChmlFrp 进行端口映射配置：

步骤1： 访问 ChmlFrp官网 注册账号
步骤2： 下载适合您操作系统的客户端程序
步骤3： 在管理面板创建隧道，配置如下：
• 本地IP：您的虚拟机IP地址
• 本地端口：22（SSH端口）
• 隧道类型：TCP
• 选择合适的中转节点

步骤4： 获取配置文件并启动frp客户端

配置完成后，您就可以通过公网IP和分配的端口号SSH连接到您的虚拟机了。
