关于 HadoopDeploy_tool

开源 · 自动化 · 可视化 · SSH免密

项目简介

HadoopDeploy_tool 是由 violet 独立开发和维护的开源项目，致力于简化 Hadoop 集群的自动化部署与管理流程。 项目集成了全自动、半自动、手动三种部署模式，支持 SSH 免密、Web 可视化、进度与日志同步、集群 Web UI 直达等特性。 适用于企业、科研和个人用户。

查看文档 GitHub
当前版本: v1.0.0

核心特性

快速部署

一键式部署 Hadoop 集群，节省 80% 的配置时间

灵活配置

支持多种配置方案，满足不同规模集群需求

安全可靠

内置安全配置最佳实践，保障集群安全

技术栈

  • Python 后端核心
  • Flask Web框架

  • Bootstrap 5 前端框架
  • Docker 容器化支持
