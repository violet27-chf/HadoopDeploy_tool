项目简介

HadoopDeploy_tool 是由 violet 独立开发和维护的开源项目，致力于简化 Hadoop 集群的自动化部署与管理流程。集成全自动、半自动、手动三种部署模式，支持 SSH 免密、Web 可视化、进度与日志同步、集群 Web UI 直达等特性，适用于企业、科研和个人用户。

查看文档 GitHub
当前版本: v1.0.0
